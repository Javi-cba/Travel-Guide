
## INSTALACION

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Javi-cba/Travel-Guide
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd Travel-Guide
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

4. Ejecuta el proyecto:
   ```bash
   npm start
   ```
