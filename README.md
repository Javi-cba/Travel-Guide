

## Funcionalidades Disponibles
La aplicación web ofrece las siguientes funcionalidades:

- **Registrarse**: Los usuarios al iniciar sesión por primera vez podrán registrarse como viajeros indicando: Tipo de Viajero, Preferencias y Presupuesto disponible.
  
- **Editar preferencias**: Una vez registrado, el usuarios pueden editar su perfil: Tipo de Viajero, Preferencias y Presupuesto. Para obtener nuevas recomendaciones de destinos. 
  
- **Ver el clima del lugar**: Los usuarios pueden consultar el clima de las recomendaciones, incluyendo la temperatura de los 7 días.

- **Ver la ubicación**: La aplicación utiliza GoogleMaps para poder mostrar la ubicación de las recomemdaciones de destino.



## INSTALACION

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Javi-cba/Travel-Guide
   ```

2. Navega al directorio:
   ```bash
   cd Travel-Guide
   cd forntend-travel
   cd backend-travel
   ```
   
3. Configura las variables de entorno:
   ```bash
   touch .env
   ```
   
4. Instala las dependencias:
   ```bash
   npm install
   ```

5. Ejecuta el proyecto:
   ```bash
   npm start
   ```
